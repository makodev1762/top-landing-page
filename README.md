# Proyecto: Landing Page

**Proyecto número:** 2 de la sección **Foundations** de **The Odin Project**

## Objetivo del proyecto (Scope)
Construir una interfaz web completa traduciendo una especificación de diseño visual a una estructura funcional de alta fidelidad. El enfoque principal es el dominio de layouts complejos y la organización visual.

### Capacidades técnicas demostradas:
* **Maquetación con Flexbox:** Posicionamiento avanzado de elementos y contenedores.
* **Fidelidad Visual:** Traducción de un diseño estático a código funcional con alta precisión.
* **Arquitectura CSS:** Implementación de estilos organizados, semánticos y reutilizables.
* **Semántica HTML5:** Uso correcto de etiquetas estructurales (`<header>`, `<footer>`, `<section>`).
* **Gestión de Activos:** Organización eficiente de imágenes y recursos mediante rutas relativas.
